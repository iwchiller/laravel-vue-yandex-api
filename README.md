## Проект Laravel+Vue для получения отзывов клиентов об объекте на Яндекс-картах через API

### Стек

- PHP 8.3+
- MySQL 5.7+
- Laravel 12+
  - sanctum 4
- Vue3 composition API
  - bootstrap 5
  - pinia 3
  -  vue-router 4
### Настройки
1. **Редактируем глобальные переменные проекта в файле `.env`:**
   ```env
   DB_CONNECTION=mysql
   DB_HOST=127.0.0.1
   DB_PORT=3306
   DB_DATABASE=laravel_vue_token_auth
   DB_USERNAME=имя_пользователя
   DB_PASSWORD=пароль_пользователя
2. Запускаем миграции БД и сидирование
3. Регистрируем своего пользователя или заходим под тестовым:
````
логин: aa@bb.com
пароль: 1234567890
````
4. Приложение содержит две страницы:

| Название   | Путь       | Функционал                                         |
|------------|------------|----------------------------------------------------|
| Parameters | /dashboard |Указываем ссылку на объект Яндекс-карт|
| Reviews    | /          |Смотрим отзывы в виде многостраничной таблицы и сводную статистику по объекту|
![](https://iw-chiller.ru/images/ya_map1.png)
![](https://iw-chiller.ru/images/ya_map2.png)
5. Виды ссылок распознаваемых приложением:
- полная ссылка, например
````
https://yandex.ru/maps/10734/kolomna/?ll=38.743978,55.093308&mode=poi&poi[point]=38.732913,55.090292&poi[uri]=ymapsbm1://org?oid=128623042633&z=14.93
````
- ссылка старого образца, например
````
https://yandex.ru/maps/org/kirovskiy_teatr_kukol_imeni_a_n_afanasyeva/1043283808/?ll=49.676576,58.601043&utm_source=share&z=15
````
- короткая ссылка, например
````
  https://yandex.ru/maps/-/CTEHFFK0
````
6. Отображение отзывов происходит по 50 на странице, это неизменямый параметр.
